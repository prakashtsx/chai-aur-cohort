# CSS

- CSS (Cascading Style Sheets) is used to style and layout web pages.
- CSS can control the color, font, spacing, and overall layout of a webpage.
- CSS can be applied in three ways:
  1. Inline CSS
  2. Internal CSS
  3. External CSS
